=====
Jacob
=====

**Якоб** - универсальный бот-помощник для студенческих групп ВКонтакте

Функционал
**********

1. Призыв студентов (отправка сообщения с упоминанием)
   
   - Опции
      - Призыв выбранных студентов
      - Призыв всех студентов
   - Содержание призыва
      - Текстовое
      - С вложениями (доступны фотографии и документы)
      - И с текстом, и с вложениями

2. Настройки

   - Настройки администратора
      - Активный чат
      - Использование имен

   - Настройки группы
      - Чат для рассылок
      - Переназначение чатов

3. Параллельная работа в нескольких чатах для одной группы
